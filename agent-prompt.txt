## User Story
As a user, I want to press `0` to temporarily deactivate selected clips (dimmed, no playback), so I can quickly A/B compare different clips without deleting them.

## Context
ACE Studio uses the `0` key to deactivate clips — they appear darkened and are skipped during playback. This is essential for non-destructive arrangement experimentation.

## Acceptance Criteria

- [ ] Deactivated clips are skipped during playback
- [ ] Right-click context menu has "Deactivate" / "Activate" option
- [ ] Deactivation state persists in project save
- [ ] Undo/redo support for deactivation toggle
